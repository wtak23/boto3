boto3.resources.action.ActionDocstring.expandtabs

ActionDocstring.expandtabs(tabsize=8)

Expands tabs to spaces

So this is a big hack in order to get lazy loaded docstring work for the help(). In the help() function, pydoc and inspect are used. At some point the inspect.cleandoc method is called. To clean the docs expandtabs is called and that is where we override the method to generate and return the docstrings.
